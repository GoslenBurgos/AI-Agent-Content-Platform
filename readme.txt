Tu equipo de creación de contenido IA, directamente en WordPress.

== Description ==

¿El cuello de botella más grande de tu estrategia digital es la creación constante de contenido original y de alta calidad? **IA Agent Content Platform** transforma tu WordPress en una central de contenido inteligente. Deja de buscar inspiración o de luchar contra la página en blanco y empieza a dirigir a tu propio equipo de especialistas en IA para que produzcan, optimicen y planifiquen tu contenido a escala.

Este plugin se integra directamente con la **API de Google Gemini**, permitiéndote aprovechar el poder de los modelos de lenguaje avanzados para automatizar tu flujo de trabajo de contenido.

== Features ==

*   **Crea tu Equipo de Especialistas IA (Gestión de Agentes):** No uses una IA genérica, construye la tuya. Define agentes con roles específicos (Experto en SEO, Copywriter, Periodista Investigador) para que cada pieza de contenido sea creada por un especialista en la materia.
*   **Contenido Siempre Fiel a tu Marca (Perfil Editorial):** Establece la audiencia, el tono y las guías de estilo de tu marca una sola vez. Todos tus agentes seguirán estas reglas, asegurando que cada artículo suene 100% a ti.
*   **Planificador de Contenido Inteligente:** Genera docenas de ideas de artículos a partir de palabras clave, evalúa su potencial de viralidad y selecciona las mejores para desarrollar.
*   **Generación de Contenido Dirigida por Agentes:** Elige un agente específico para escribir un artículo completo basado en un título y un tema, siguiendo el perfil editorial que definiste.
*   **Calendario de Contenido Visual:** Planifica y visualiza toda tu estrategia de contenido en un calendario interactivo. Reprograma posts con una simple acción de arrastrar y soltar (drag-and-drop).
*   **Amplifica tu Mensaje (Planificador de Redes Sociales):** Genera automáticamente textos atractivos para redes sociales basados en tus artículos y programa su publicación.
*   **Historial y Versionado de Contenido:** Accede a un historial completo de las versiones de tu contenido. Restaura una versión anterior con un solo clic si una edición no te convence.
*   **Analíticas de Rendimiento:** Realiza un seguimiento de las vistas de tus artículos publicados directamente desde el panel de contenido para medir su impacto.

== Installation ==

1.  **Sube el plugin:**
    *   Descarga el archivo `.zip` del plugin.
    *   En tu panel de WordPress, ve a `Plugins > Añadir nuevo > Subir plugin`.
    *   Selecciona el archivo `.zip` y haz clic en `Instalar ahora`.
2.  **Activa el plugin:**
    *   Una vez instalado, haz clic en `Activar plugin`.
3.  **Configura la API Key:**
    *   Ve a `IA Agent Platform > Settings`.
    *   Introduce tu clave de la API de Google Gemini.
    *   ¡Listo! Ya puedes empezar a crear tu equipo de agentes.

== Frequently Asked Questions ==

= ¿Qué necesito para usar este plugin? =

Necesitas una clave de la API de Google Gemini. Puedes obtenerla en Google AI Studio.

= ¿Puedo usar diferentes modelos de Gemini? =

¡Sí! En la página de `Settings`, puedes especificar el nombre del modelo que deseas utilizar (por ejemplo, `gemini-1.5-flash-latest` o `gemini-1.5-pro-latest`).

= ¿Puedo crear mis propios agentes? =

Sí, esa es una de las características principales. Puedes definir tantos agentes como necesites, cada uno con su propio rol, experiencia y prompt personalizado.

= ¿El plugin afecta la velocidad de mi sitio público? =

No. Toda la lógica del plugin, las llamadas a la API y los scripts se cargan únicamente en el área de administración de WordPress (`/wp-admin`). No añade ninguna carga a la parte pública de tu sitio, a excepción del contador de vistas, que es extremadamente ligero.
